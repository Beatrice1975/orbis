ITALIANO

Orbis
Orbis è una collezione di animazioni HTML5 progettate per essere visualizzate su uno schermo di smartphone e riflesse all'interno di un oggetto semitrasparente stampato in 3D. L'effetto ottico crea un'illusione di profondità tridimensionale, trasformando il gioco di luci in esperienze visive immersive e rilassanti.
Come funziona
Apri una delle animazioni sul tuo smartphone
Appoggia il telefono su una superficie piana con lo schermo verso l'alto
Posiziona l'oggetto stampato in 3D sopra lo schermo
Guarda la magia: la luce si diffonde sulle pareti dell'oggetto creando effetti 3D spettacolari

Menu Principale
Apri index.html per accedere al menu di selezione con tutte le animazioni disponibili.

Caratteristiche Tecniche
Nessuna dipendenza esterna — funziona offline
Responsivo — si adatta a qualsiasi dimensione di schermo
Canvas API — rendering veloce e fluido

Codice QR
Scansiona il codice QR stampato sul supporto 3D per accedere direttamente al menu Orbis dal tuo smartphone.

INGLESE

Orbis
Orbis is a collection of HTML5 animations designed to be displayed on a smartphone screen and reflected inside a semi-transparent 3D-printed object. The optical effect creates an illusion of three-dimensional depth, transforming the play of light into immersive and relaxing visual experiences.
How it works
Open one of the animations on your smartphone.
Place the phone on a flat surface with the screen facing up.
Position the 3D-printed object over the screen.
Watch the magic: the light diffuses across the walls of the object, creating spectacular 3D effects.

Main Menu
Open index.html to access the selection menu with all available animations.

Technical Features
No external dependencies — works offline
Responsive — adapts to any screen size
Canvas API — fast and smooth rendering

QR Code
Scan the QR code printed on the 3D stand to access the Orbis menu directly from your smartphone.
